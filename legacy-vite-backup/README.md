# MLBuilder Legacy Vite Backup

This directory contains the original, pre-migration Vite snapshot of MLBuilder.

## Contents
- `src/` - Pure React components, page files, hooks, contexts, and style definitions.
- `vite.config.ts` - Vite bundler configuration.
- `index.html` - HTML entry point.
- `package.json` - Vite dependencies.
- `tsconfig.json` - Original TypeScript configuration.

## Usage
Reference only. Once the Next.js migration is verified as fully working and deployed, this directory can be safely deleted.
