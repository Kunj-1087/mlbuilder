# LinkedIn Job Scraper (MLBuilder Automation Project)

A complete Python automation that monitors LinkedIn for new job postings matching your search criteria, scrapes the details, and exports everything to a clean CSV file.

## Setup Instructions

1. Install Python 3.8+ on your machine.
2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the entrypoint script:
   ```bash
   python main.py
   ```

*Note: For the actual automation, make sure to set up ChromeDriver or appropriate Selenium web drivers on your path.*
